# Editor de Texto
## Como usar?
Esse editor de texto foi feito para ser usado para desenvolver projetos de Python, HTML, CSS e Javascript. Depois, você pode levar uma pasta zipada com o projeto que você criou
## Considerações
Usei as bibliotecas JSZip e CodeMirror para, respectivamente, zipar o projeto e fazer um editor com destaque de sintaxe, indexação, etc.(Este último deixa o editor parecido com o do VS Code) --> *Parecido, não quer dizer igual*
